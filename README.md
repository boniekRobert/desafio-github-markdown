# 🎬 PopcornList: Guia Colaborativo de Entretenimento

> Desafio prático de Git, GitHub e Markdown para o cursinho de GitHub DIO.

Este repositório foi criado para listar, organizar e avaliar **filmes, séries e jogos favoritos**. O principal objetivo deste projeto é treinar comandos do Git (como clone, commits e pushes) e fluxos de trabalho do GitHub (como gerenciamento de colaboradores, Issues e Pull Requests) usando a formatação **Markdown**.
